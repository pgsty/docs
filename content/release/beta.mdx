**Note:** There is currently no active Beta version available, as the [latest stable release](https://www.google.com/search?q=/release/latest), [v3.7.0](https://www.google.com/search?q=/release/note#v370), has just been published.

You can get the latest beta version of Pigsty with:

```bash tab="Default"
curl -fsSL https://repo.pigsty.io/beta | bash; cd ~/pigsty
```
```bash tab="Mirror"
curl -fsSL https://repo.pigsty.cc/beta | bash; cd ~/pigsty   # china mirror
```
