## Quick Start

[**Single node installation**](/docs/install): [Prepare](/docs/prepare) an [ssh-accessible](/docs/prepare/admin#check-accessibility) [node](/docs/prepare/hardware#node) with fresh installed [Linux](/docs/prepare/linux),
 run as [user](/docs/prepare/admin#user) with nopass [`ssh`](/docs/prepare/admin#ssh) and [`sudo`](/docs/prepare/admin#sudo-privileges) privileges:

<Steps>
<Step>

[**Download**](/docs/prepare/software#pigsty) pigsty with:

```bash
curl -fsSL https://repo.pigsty.io/get | bash; cd ~/pigsty;
```

</Step>
<Step>
[Bootstrap](/docs/install/offline#bootstrap) pigsty: (install [ansible](/docs/admin/ansible) dependency)

```bash
./bootstrap
```
</Step>

<Step>
    [**Configure**](/docs/config/configure) the [inventory](/docs/config/inventory) file according your need. (<span className={"text-red-500 font-black"}>CHANGE PASSWORDS</span> in `pigsty.yml`)

```bash
./configure
```

</Step>
<Step>

[**Install**](/docs/install/singleton) everything according to your config:

```bash
./install.yml
```
</Step>


</Steps>


Pigsty Installed!  Explore the [**User Interface**](/docs/install/interface), check port `5432` for [Postgres](/docs/pgsql) [Service](/docs/pgsql/service), port `3000` for grafana dashboards (`admin` / `pigsty`).
